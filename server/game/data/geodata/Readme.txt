##############################################
ГЕОДАТА — как установить
##############################################

Файлы геодаты (*.l2j) в git НЕ хранятся (см. .gitignore в корне репозитория),
поэтому эту папку нужно один раз наполнить на сервере.

--------------------------------------------------
Автоматически (рекомендуется)
--------------------------------------------------
Из корня репозитория выполни:

    bash server/game/download_geodata.sh

Скрипт скачает 215 регионов (~650 МБ) формата .l2j в эту папку.

--------------------------------------------------
Источник и совместимость
--------------------------------------------------
* Источник : https://github.com/bartty/mobius-geo  (геодата под L2J Mobius)
* Формат   : .l2j, 256x256 блоков (flat / complex / multilayer), имена вида
             XX_YY.l2j (например 20_18.l2j) — так их ищет движок
             (GeoEngine, шаблон "%d_%d.l2j").
* Проверено: все 215 файлов читаются штатным парсером Region(ByteBuffer) из
             GameServer.jar без ошибок (буфер прочитан ровно до конца).

--------------------------------------------------
После установки
--------------------------------------------------
1. В server/game/config/GeoEngine.ini должно быть  PathFinding = 2
   (значение по умолчанию в репозитории).
2. Перезапусти игровой сервер:  bash stop_linux.sh && bash start_linux.sh
3. Если геодата ещё НЕ скачана — временно поставь PathFinding = 0,
   иначе сервер будет лагать и писать ошибки поиска пути в логи.

Требования по памяти: с геодатой игровому серверу нужно больше heap.
В этой сборке heap уже задан в server/game/java.cfg (-Xmx8g -Xms2g) — этого
с запасом хватает. Менять не нужно.
