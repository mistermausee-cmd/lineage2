Объекты крепости активированы.
Плата за использование каждого объекта будет автоматически списываться со счёта склада клана за каждый период времени. Если на счету не останется средств, соответствующий объект будет сброшен.
Чтобы проверить оставшееся время для каждого объекта, используйте меню Управление крепостью.
