Помощь по панели:
Введение
Верхняя боковая панель содержит основные области администрирования. Эта панель присутствует на пяти главных страницах, на которые она указывает.
Команды от //admin1 до //admin5 также перенесут вас прямо в соответствующие области администрирования.
Как правило, кнопки команд, которыми вы пользуетесь чаще, должны располагаться первыми.
Когда кнопке команды требуется один или несколько параметров, их следует вписать в Квикбокс через пробел.
Некоторые кнопки работают над вашей целью, другие учитывают только параметры, а третьи требуют и то, и другое.
Чаще всего команды, зависящие от квикбокса, располагаются ближе к нему.
На каждой важной странице есть Квикбокс. Кнопки помощи содержат контекстные подсказки — попробуйте их все ;)
Когда определённые команды вызываются с неверной целью и/или неверными/отсутствующими параметрами, всплывают диалоги, связанные с действием.
Другие же команды выведут руководство по использованию, отправив вам системные сообщения — убедитесь, что ваш клиент настроен на отображение таких сообщений!
