Кнопки панели эффектов
 Здесь собраны игровые спецэффекты: землетрясения, смена неба, необычные анимации персонажей и мобов.
1. Первый набор эффектов.
Para: Парализовать персонажа (цель/имя).
UnPara: Снять эффект паралича (цель/имя).
ParaAll: Парализовать всех персонажей в мире, кроме вашего. Используйте с умом — раздражает людей и вызывает лаги.
UnParaAll: Снять паралич со всех онлайн-персонажей.
Effect: Заставить персонажей проиграть анимацию каста. В квикбоксе укажите skill_id, (опц.) уровень умения и (опц.) длительность.
Social: Заставить персонажей проиграть социальные анимации. В квикбоксе можно указать socialid, (опц.) имя игрока и (опц.) радиус. Если указан только один параметр — затрагивает цель.
Abnormal: Изменить аномальное состояние персонажей. В квикбоксе можно указать битовую маску аномалии, (опц.) имя игрока и (опц.) радиус. Если указан только один параметр — затрагивает цель.
Transform: Выполнить трансформации камаэлей. В квикбоксе должен быть transformId; затрагивает цель.
2. Второй набор эффектов (меньше зависит от цели).
Day/Night Sky: Смена неба. Затрагивает весь мир.
Dawn/Dusk Sky: Устанавливает небо Семи Печатей.
Red Sky: Устанавливает эффект неба Проклятого Оружия.
Big/Shrink Head: Быстрый доступ к аномалии 0x2000. Затрагивает только цель. Подробнее см. Abnormal.
Quake: Вызывает землетрясение. В квикбоксе укажите интенсивность и длительность.
SetName: Временно меняет имя персонажа.
Sounds: Открывает меню с эмбиент-звуками для проигрывания.
