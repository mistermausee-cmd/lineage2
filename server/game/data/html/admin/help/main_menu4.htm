Кнопки главной панели
Способы изменить судьбу вашей цели.
Kill: убивает цель; опционально может брать из квикбокса радиус или имя игрока.
Res: воскрешает цель, игрока по имени или игроков в указанном радиусе.
Open: Если ваша цель — дверь, она откроется по вашей команде.
Close: Если ваша цель — дверь, она закроется по вашей команде.
Recall NPC: Если ваша цель — НПС/моб, отныне он будет спавниться в ваших текущих координатах с вашим направлением.
Delete: Если ваша цель — НПС/моб, он больше не будет спавниться.
Heal: Как //kill или //res, берёт цель, имя или радиус и разом восстанавливает HP/MP/CP до максимума.
Full Food: Полезно для забагованных питомцев — позволяет питомцу восстановить уровень сытости и корректно быть отозванным.
