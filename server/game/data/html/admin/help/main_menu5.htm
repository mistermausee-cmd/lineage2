Кнопки главной панели
Приятные мелочи, дающие почувствовать себя особенным.
List ON: ваше имя появится в результатах пользовательской команды /gmlist.
List OFF: у вас есть дела поважнее поддержки петиций. Ваше имя не будет в списке поддержки.
Diet: Переключатель, позволяющий носить большие объёмы вещей без штрафов за вес.
Invis: Переключатель режима невидимости.
Petitions: Просмотр списка петиций.
Silence: Переключатель режима приёма личных сообщений.
TradeOff: Переключатель режима отказа от торговли.
Invul: Переключатель режима неуязвимости.
Wyvern: Оседлайте виверну, притворитесь владельцем замка, покрасуйтесь.
Strider: Оседлайте страйдера, притворитесь, что у вас есть обученный питомец.
Enchant: Открывает диалог заточки предметов. Затрагивает цель.
GM Shop: Лень создавать предметы по ID? Получите их здесь бесплатно.
