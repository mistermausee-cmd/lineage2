Кнопки панели сервера
 Здесь собраны действия, связанные с сервером: настройки, множители, кэш.
1. Перезагрузка и прочее.
Reload: Пытается перезагрузить указанный менеджер. Возможные значения: multisell,skill,npc,htm,item,instancemanager,npcwalkers.
Reload HTM: Очищает кэш HTM.
Reload File: Очищает кэш для указанного HTM-файла.
Skill Test: Как и //effect, принимает ID умения как параметр; затрагивает цель или кастера.
Reload CW: Перезагружает менеджер проклятого оружия.
Reload Zone: Перезагружает менеджер зон.
Reload Crest: Пересобирает кэш гербов.
Crest Fix: Конвертирует старые файлы гербов, если найдены.
Reload Tele: Перезагружает точки телепортации.
Social: Заставить персонажей проиграть социальные анимации. В квикбоксе можно указать socialid, (опц.) имя игрока и (опц.) радиус. Если указан только один параметр — затрагивает цель.
Open/Close All: Открыть или закрыть все двери на сервере.
Zone check: Выполняет проверку зон по запросу.
2. Геоданные.
(те, кто знает, что делает, также знают, какая из них верная)
3. Разработка.
FightCalc: Запускает диалог расчёта боя между двумя мобами. Packet Forge: Запускает средства конструирования пакетов.
System message: Отправляет вам указанное системное сообщение.
4. Обслуживание.
Stats: Показывает статистику сервера: использование памяти, задачи и т.д.
Login: Показывает информацию и параметры, связанные с логин-сервером.
Shutdown: Показывает действия по перезапуску/выключению сервера.
