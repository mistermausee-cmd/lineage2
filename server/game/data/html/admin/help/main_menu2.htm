Кнопки главной панели
Здесь собраны самые полезные команды.
То, что вы будете использовать чаще всего
Item: берёт id предмета и, опционально, количество из квикбокса, чтобы создать такой предмет в вашем инвентаре. Показывает специальный диалог, если ничего не указано.
Teleport: читает координаты из квикбокса. Показывает меню телепортов и перемещений, если ничего не указано.
Spawn: берёт npc_id или имя (пробелы заменяются подчёркиваниями); опционально количество и время респавна. Показывает специальный диалог, если ничего не указано.
List Spawns: берёт npc_id или имя (пробелы нужно заменить подчёркиваниями). Показывает результаты системными сообщениями.
Announce: берёт текст объявления из квикбокса; показывает меню объявлений, если текст не указан.
Set: требует "параметр=значение" в квикбоксе. Показывает диалог настроек, если ничего корректного не указано.
GM Chat: транслирует содержимое квикбокса только для ГМ. После трансляции всплывёт меню админа.
List+Go: Как кнопка «Мне повезёт» в Google — телепортирует вас к первому результату команды List Spawn.
Go To: читает имя игрока из квикбокса. Показывает экран управления персонажами, если ничего не указано.
Recall: призывает игрока к вашей позиции, если имя игрока указано в квикбоксе
Karma: задаёт карму цели равной значению из квикбокса (если цель — игрок, естественно).
No Karma: работает над целью как кнопка Karma, но игнорирует значение квикбокса, обнуляя карму.
