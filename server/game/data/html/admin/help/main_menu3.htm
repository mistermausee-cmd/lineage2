Кнопки главной панели
Быстро разбирайтесь с глупостями игроков.
ChatBan: может брать имя игрока и длительность из квикбокса, либо только длительность для бана чата цели.
ChatUnBan: цель или игрок по имени освобождаются от бана чата.
Jail: Работает как Chat Ban по параметрам, но сажает персонажа в тюрьму. Если длительность не указана, наказание вечное.
UnJail: Выпускает игроков из тюрьмы, они появляются в Деревне Флоран. Оффлайн-игроки тоже затрагиваются.
Kick: отключает игрока от сервера.
Ban: По имени персонажа в квикбоксе банится весь его аккаунт.
UnBan: По имени игрока его аккаунт разбанивается.
Find: Показывает список имён персонажей, совпадающих с указанным, либо диалог поиска, если ничего не указано.
Repair: некоторые значения для указанного персонажа будут возвращены к безопасным значениям по умолчанию. Используйте с умом!
