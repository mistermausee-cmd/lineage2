Кнопки панели игры
 Здесь собраны игровые аспекты: персонажи, кланы, проклятое оружие и статичные события.
1. Игроки под вашим управлением.
Manage: Старая панель типовых действий над игроком.
List: Диалог со списком онлайн-игроков; показывает уровень, класс и поле поиска. Кликните по имени для подробностей.
Set Name: Быстрая перманентная смена имени игрока. Предназначено для соблюдения правил, а не для событий.
Set Title: Быстрая перманентная смена титула. Игроки могут сменить его позже, если они нобли или состоят в клане.
2. Кланы: Центр искусственного общества.
Create: целевой персонаж станет лидером. Имя клана нужно указать в квикбоксе.
Delete: разделяй и властвуй. Мгновенно распускает клан по выбранному лидеру.
Set Level: Укажите уровень от 0 до 8. Цель должна быть лидером клана.
Reputation: Укажите положительное или отрицательное значение для изменения репутации. Цель должна быть лидером клана.
Castles/CH: Диалоги управления замками, осадами и клан-холлами.
Info: Показывает окно информации о клане. Цель должна быть лидером клана.
3. Проклятое оружие. Хаос по меню.
Info: отображает информацию о состоянии всех поддерживаемых проклятых оружий.
Add: если ваша цель — персонаж, он получит указанное оружие. В квикбоксе нужно указать id предмета, имя предмета или его часть (напр.: zariche)
Remove: убирает проклятое оружие из мира. В квикбоксе нужно указать id предмета, имя предмета или его часть (напр.: akamanah)
4. Семь Печатей и странствующие гномы.
Blacksmith: Выводит координаты и телепортирует к нему, если он заспавнен.
Merchant: Выводит координаты и телепортирует к нему, если он заспавнен.
Respawn: Показывает время до респавна системным сообщением.
5. Олимпиада: в разработке.
Save Olympiad: Вручную сохраняет состояние Олимпиады.
Manual Hero: вычисляет победителей и награждает новых героев.
