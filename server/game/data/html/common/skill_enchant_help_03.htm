Существует несколько вариантов заточки умений: Заточка, Благословлённая Заточка, Возврат и Смена Пути Заточки.
Заточка Умения
Этот способ заточки требует 1 [Высший Кодекс Гиганта] и некоторое количество Адены. Если попытка не удастся, уровень заточки сбрасывается до нуля.
Благословлённая Заточка

Этот способ требует 1 Высший Кодекс Гиганта: Мастерство и в 5 раз больше Адены, чем при обычной заточке умения. Если ваша попытка не удастся, значение заточки сохраняется, что делает этот способ сравнительно безопасным для заточки сильно заточенного умения.
Возврат

Этот вариант требует 1 Высший Кодекс Гиганта: Забвение и понижает заточку умения на один уровень за раз. Если применить его, когда умение на ур. 1, умение будет сброшено до нуля.
Смена Пути Заточки

Эта функция меняет тип заточки, применённый к умению. Помимо Высшего Кодекса Гиганта: Дисциплина, эта процедура требует 20% Адены, необходимой для следующего уровня заточки.
