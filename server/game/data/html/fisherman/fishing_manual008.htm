Пособие по рыбалке:
Рыболовные заряды, созданные Гильдией Чёрной наковальни по просьбе Ампитри, основателя Гильдии рыбаков, представляют собой разновидность зарядов душ, сделанную для рыбалки.
Но если заряды душ усиливают удар, то рыболовные заряды повышают шансы поймать рыбу.
Это предмет, который поможет тебе ловить рыбу эффективнее.
