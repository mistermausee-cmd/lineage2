Пособие по рыбалке:
Существует три вида наживки.
Это обычная наживка, наживка R99, которую могут использовать персонажи 99-го уровня и выше, и наживка PC-кафе, которую можно приобрести за очки PC-кафе.
Из них Гильдия рыбаков продаёт обычную наживку и наживку R99.
С каждым видом наживки можно ловить разные виды рыбы. Условия покупки и использования у видов наживки тоже различаются. Могучую рыбу, самую дорогую рыбу, которую можно поймать, ловят с помощью наживки R99 или наживки PC-кафе.
