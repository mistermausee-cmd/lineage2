Секретный отчёт Гильдии:
Автор: Пириэль Аурура, коллекционер высшего ранга Гильдии Бронзового Ключа
(Пропуск)...Человеческие мистики классифицируют Иконокласиса как духа света. Жрецы Эйнхасад называют их ангелами. Предполагается, что когда Скрижаль Видения была создана посредством череды ритуалов, они были призваны из иного мира, чтобы охранять её...
(Пропуск)... Во время своего паломничества я несколько раз встречал Иконокласиса и обнаружил несколько особенностей этого духа. Основываясь на своих личных знаниях, я привожу следующие рекомендации для членов гильдии, которым в будущем может доведётся столкнуться с Иконокласисом.
