Наставник может одновременно иметь до 3 учеников. Каждый раз, когда ученик повышает уровень, наставник получает по почте *Метку Ученика*.
С 1 по 50 уровень наставник получает письмо каждые 10 уровней. После 50 уровня наставник получает письмо каждый раз, когда ученик повышает уровень. Обязательно удаляйте ненужные письма, так как вы не сможете получать новые, если почтовый ящик переполнен.
Когда наставник и ученик оба в сети, наставник может получить мощное усиление, повышающее характеристики, а ученик — опыт, SP и бонусное усиление.
Когда ученик завершает пробуждение или 3-е освобождение, он автоматически выпускается и получает по почте Сертификат Ученика. Если он отнесёт его Наставнику-Проводнику, тот обменяет его на Диплом, который можно использовать для покупки различных предметов.
Если наставник и ученик расторгают договор досрочно, наставник должен ждать 2 дня, чтобы найти нового ученика. Если ученик выпускается, наставник должен ждать 1 день, чтобы найти нового ученика.
