Наставник может накладывать сильные усиления на своего ученика.
Однако, если наставник и ученик находятся слишком далеко друг от друга, применить это умение будет сложно. Чтобы решить эту проблему, у ученика есть умение призыва наставника.
Обратитесь к категориям умений клана / героя / наставничества в окне умений.
