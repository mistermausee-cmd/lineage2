Вестник Гильдии:
Хранитель одной из Скрижалей Видения. Всегда настаивает на поединке один на один. Если его противник нарушает правила поединка один на один, он выйдет из боя, даже если побеждает.
Защищаясь, он воздвигает оборонительную железную стену. В этом состоянии он неуязвим. Попробуйте спровоцировать его умением Ненависть.
