Вестник Гильдии:
Архангел Иконокласис: Хранитель одной из Скрижалей Видения. Он всегда настаивает на поединке один на один. Если его противник нарушает правила поединка один на один, он выйдет из боя, даже если побеждает.
Он атакует взрывной сферой, содержащей его собственную энергию. С этой сферой следует разобраться в первую очередь при любой атаке на него.
