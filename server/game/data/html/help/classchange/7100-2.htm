Важность родословной:
Орки проводят магический ритуал, вливая кровь более могущественных существ в собственные тела, чтобы усилить себя.
Однако, когда два разных типа крови смешиваются, организм, принявший эту смесь, коренным образом меняется.
Например, Люди, созданные Гран Каином, по своей структуре отличаются от всех четырёх других рас.
Известны случаи, когда это изменение удавалось обратить, используя магию Оглушения и умение, удаляющее смешанную кровь из вен.
