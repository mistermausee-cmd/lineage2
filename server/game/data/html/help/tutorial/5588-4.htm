Радар
Красная точка на радаре обозначает ваше текущее положение, а жёлтая точка на краю радара — точку назначения, заданную вам NPC.
Буква N на радаре указывает направление на Север. Изменение направления персонажа соответственно поворачивает радар. Поэтому направление, куда смотрит персонаж, всегда будет вверху радара.
