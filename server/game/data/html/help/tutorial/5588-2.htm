Точка обзора (POV)
Перемещение мыши при зажатой правой кнопке меняет точку обзора вашего персонажа. Такое гибкое изменение обзора особенно полезно во время охоты, когда персонажу нужно осматриваться в поисках монстров.
Щёлкните правой кнопкой мыши ещё раз, и обзор по умолчанию восстановится.
Нажатие клавиш Page Up/Page Down также позволяет изменить базовую точку обзора персонажа.
