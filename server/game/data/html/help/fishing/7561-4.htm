Руководство по рыбалке:
Созданные Гильдией Чёрной Наковальни по просьбе Ампитри, основателя Гильдии Рыбаков, Рыболовные заряды — это разновидность Заряда Души, предназначенная для рыбалки.
Но если Заряд Души усиливает удар, то Рыболовные заряды повышают шанс поймать рыбу.
Это предмет, который поможет вам ловить рыбу эффективнее.
