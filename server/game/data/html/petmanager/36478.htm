С помощью функции восстановления вы можете превратить питомца, прошедшего эволюцию в особую форму, обратно в его исходную форму. Поскольку при этом переносится значение опыта, восстановление питомца высокого уровня может фактически привести к снижению его уровня.
Любые предметы из инвентаря питомца будут перенесены в инвентарь владельца.
