Помощник Подземного колизея:
В игре могут участвовать только отряды, в которых более 7 участников, и, если они побеждают, продолжать играть. Вы также можете участвовать только на арене, отведённой для вашего уровня.
Когда вы входите на арену, вы увидите Башню воскрешения для каждой команды. Пока вы не уничтожите башню противников, она будет продолжать их воскрешать. Осмотритесь на арене, чтобы понять, как это повлияет на бой. Это ведь больше, чем просто схватка, знаете ли...
Помните также, что любое усиление, наложенное на команду, будет отменено, если один из участников не соответствует требованиям по уровню. Ещё один совет: подумайте о том, чтобы поставить на передовую того участника команды, кто лучше всех сражается в одиночных поединках.
Итак — готовы доказать, насколько вы на самом деле хороши?
