Гранд-мастер Золдарт:
Я Гранд-мастер Золдарт из Гильдии Камаэлей. Ба, да ты выглядишь измотанным! Позволишь дать тебе совет? Для Камаэля, желающего сменить профессию, важно обладать разными способностями, но крепкая физическая сила — важнейший фактор. Тебе придётся совершить немало трудных странствий, чтобы обрести смену профессии, — как же ты их переживёшь, если ты уже слаб, как новорождённый жеребёнок? Не растрачивай силы — прибереги их на тот случай, когда они понадобятся.
