Вестник Обновлений:
Рекомендуется искателям приключений 81 уровня и выше. Мстительные духи вождей, спавшие в Местах Погребения Позора, были освобождены, когда Варангка в Логове Зла обрёл силу Каши и воскресил их вместе с их последователями с помощью силы Каши.
Теперь их очень много, поэтому, если пойдёте, обязательно делайте это в сильной группе.
На самом деле Расхитители Гробниц, которые прежде хозяйничали в этом месте, теперь слишком напуганы, чтобы даже покидать своё укрытие! По крайней мере, одной заботой меньше...
