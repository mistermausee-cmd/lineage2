Вестник Обновлений:
Рекомендуется искателям приключений 80 уровня и выше. Зал Иллюзий — это новое измерение, открытое Следопытами, изучавшими Камароку.
Существующий в пространстве снов, он, по слухам, полон монстров, рождённых из кошмаров и хаоса, под предводительством Энкинеля, Захватчика Снов.
Поскольку Зал Иллюзий физически не существует в нашем мире, вы можете попасть туда только с помощью Следопыта.
Следопыты есть в каждой деревне, так что найти одного не составит труда.
