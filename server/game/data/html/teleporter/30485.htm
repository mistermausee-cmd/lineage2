Маг Башни Слоновой Кости Белкадхи:
В прошлом Башня Слоновой Кости отправляла сюда множество магов-женщин. Думаю, многие здесь не воспринимали их всерьёз, и... что ж, скажем так, отношения испортились. Некоторые из древних руин были разрушены в последовавших сражениях, и теперь сюда дозволено присылать только магов-мужчин.
Скажу тебе, обстановка здесь сейчас нестабильна. Если хочешь продолжить, тебе стоит сначала как следует подготовиться.
