Хранитель Портала Верона:
Приветствую тебя, искатель знаний! Добро пожаловать в Башню Слоновой Кости, почтенное хранилище мистических тайн. Я Верона, Хранитель Портала Башни.
Башня Слоновой Кости состоит из нескольких этажей, каждый из которых полностью отделён от остальных. Чтобы перемещаться с одного этажа на другой, ты должен воспользоваться услугами хранителя портала вроде меня.
Сейчас ты стоишь в холле первого этажа. В подвале расположен торговый центр, Гильдия Мистиков Людей — на 2-м уровне, Гильдия Эльфийских Мистиков — на 3-м, а Гильдия Тёмных Мистиков — на 4-м. Куда ты хочешь отправиться?
