Кицка:
Большая Кость Дракона -- это предмет для призыва могучего дракона.
В Долине Драконов есть 4 Драконьих Вихря. Если использовать предмет там, можно призвать одного из следующих рейдовых монстров: Изумрудный Рог, Пылевой Всадник, Кровавая Муха, Крыло Чёрного Кинжала, Призыватель Теней, Шипастый Резак, Мускульный Бомбардир.
Монстры, с которых выпадает: в Долине Драконов это Стражи Дракос, Воины Дракос, Охотники Дракос, Убийцы Дракос, Гигантский Ядовитый Скорпион, Крепкий Ядовитый Скорпион, Ядовитый Скорпион, Драки-Нетопыри, Паразитические Пиявки, Изумрудные Драки, Голодные Паразитические Пиявки, Самоцветные Драконы, Драконьи Следопыты Долины, Драконьи Разведчики Долины, Следопыты Песчаного Драка и Следопыты Пылевого Дракона.
