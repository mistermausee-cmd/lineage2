Страж Зером:
Тропа Смерти? Это широкая дорога между Долиной Драконов на севере и Подковообразным горным хребтом Диона. С географической точки зрения это единственный путь, соединяющий Орен и Аден. Однако по всей её длине бродят опасные чудовища, так что путешествовать по ней без большого отряда опасно.
