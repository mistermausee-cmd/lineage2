Часовой Кенделл:
На первый взгляд этот лес может показаться мирным, но внутри таятся ужасные враги. Сотни лет наше королевство подвергалось нападениям Орков с Туманных Гор, грибка из Моря Спор и гигантских пауков из Пустоши Глудио.
Странствуя по лесу, ты ни на миг не должен забывать об угрозах, что прячутся за прелестными пейзажами.
