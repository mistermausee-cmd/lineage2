Часовой Трионелл:
Нейтральная Зона начинается сразу за этим мостом. После войны между Эльфами и Тёмными Эльфами эту область объявили Нейтральной Зоной, чтобы предотвратить столкновения. Долгое время она служила буферной зоной.
Поскольку ни одна из сторон не заходит в эту зону, лес здесь густ, и сюда стекаются всевозможные диковинные существа, которых не встретишь больше нигде.
