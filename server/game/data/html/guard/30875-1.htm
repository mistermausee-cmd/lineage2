Страж Карлтон:
Есть тайрим... Сгусток огня, плывущий по воздуху... И нежить, что зовётся столь громко — болотный судья... Есть гигантские крысы, называемые тулбенами, а о горгульях ты ведь знаешь, не так ли? Болотные преследователи тоже очень опасны. Но самый зловещий из всех — болотный дрейк. С виду он как любой другой дрейк, но атакует пламенем...
