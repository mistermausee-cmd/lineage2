Генерал Дилиос:
Семя Бесконечности опаснее, чем когда-либо прежде. Прежние планы не сработают — мы думаем войти командным каналом из 5 групп.
Как только мы войдём, нам нужно будет пробраться в Зал Страданий, охраняемый проклятыми близнецами Йехан. Победив этих рыцарей, ты войдёшь в Сердце Бесконечности, где ждёт Экимус.
Как только мы одолеем Экимуса, Грасия вновь обретёт толику покоя, по крайней мере на время.
