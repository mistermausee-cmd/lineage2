Заряды Души — это сгустки духовной силы, изготовленные искуснейшими гномьими мастерами. Приложенные к оружию, они добавляют изрядное количество урона, знаешь ли!
Чтобы их опробовать, просто перетащи заряд(ы) души в окно быстрого доступа, а затем нажми правую кнопку мыши, чтобы начать ими пользоваться.
