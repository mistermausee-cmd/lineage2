Привратник Ипподрома:
Заходи! Добро пожаловать на Ипподром Гонок Монстров!
Владелец ипподрома достиг соглашения с Башней Слоновой Кости, что позволяет попадать на Ипподром Гонок Монстров через привратников во всех крупных городах. Чтобы защитить клиентов и их деньги, проверки для предотвращения проникновения посторонних были усилены.
Заходи внутрь и осмотри новый Ипподром Гонок Монстров! Веселись!
