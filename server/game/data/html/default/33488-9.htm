Акрам:
Feoh Волшебник Солткриг был одним из Семи Мудрецов-Великанов. Солткриг, в высшей степени мудрый и разумный Великан, был стратегом-аналитиком. Он множество раз вырывал победу из когтей поражения благодаря своему блестящему планированию. С юных лет его завораживали фундаментальные стихии природы. За годы изучения он стал главным знатоком четырёх стихий и теории маны.
