Старейшина Вердуры Эликия:
Этот Командный Пост — это место, где правая рука Кельбима, командир Бернстайн, обучает новобранцев.
Насколько я слышал, внутри проводят чудовищные опыты. Аденский авангард ведёт расследование внутри, так что, уверен, вскоре мы узнаем больше.
Врата командного поста почти всё время закрыты. Единственный раз, когда они открываются происходит, когда Бурштейн пытается проникнуть в Командный пост.
Именно тогда Девианна, которая находится рядом со входом в Командный Пост использует свою магию, чтобы удерживать ворота открытыми.
Это всё, что нам удалось выяснить.
