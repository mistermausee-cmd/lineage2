Исследователь Питгон:
Это устройство, созданное древними для быстрого перемещения внутри башни. С его помощью ты можешь мгновенно перенестись на другой уровень этой башни, но, чтобы активировать механизм, тебе понадобится Камень Измерения. Если ты хочешь попасть на Средний уровень, тебе понадобится Синий Пространственный Камень. Если ты хочешь отправиться на Верхний уровень, тебе понадобится соответствующий Зелёный Пространственный Камень.
Проблема в том, что их действительно трудно добыть, особенно Красный Камень Измерения - им владеют племена Платины и ангелы. Может, лестница будет для тебя лучшим вариантом?
