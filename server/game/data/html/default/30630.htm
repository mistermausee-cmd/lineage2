Орим из Тени:
Меня зовут Орим, и я служу великому Волшебнику Хардину как моему господину. Я ищу истину в этой заброшенной земле, где лежат тёмные тени земляного дракона Антараса. Если всё, что остаётся на свету, которым большинство восхищается, есть лицемерие, то, возможно, истина и вправду сокрыта во тьме.
