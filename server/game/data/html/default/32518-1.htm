Вортекс:
Вихрь смятения... Одна часть... Таинственная сила...
Воздушная Расщелина... Это поле битвы, куда можно попасть через небольшой вихревой зал, испускающий таинственную энергию. При регистрации ты можешь выбрать красную или синюю команду. Когда число участников каждой команды превышает 9, поле битвы автоматически становится активным.
Чтобы выйти на поле битвы, ты должен уметь превращаться в Летающий Трансформированный Объект Участники каждой команды получают очки, атакуя персонажей другой команды или уничтожая компрессор, расположенный на поле битвы. Если энергия сосредоточена на одном участнике, он становится целью сосредоточенной атаки (ЦСА). Повергнув эту цель, ты можешь получить бонусные очки.
Команда-победитель получает куски наждака и звёздные камни. Соедини их с энергией семени, и с помощью созданного преобразователя ты сможешь превратить их в мощные кристаллы атрибутов.
