Мы тоже не уверены, поэтому полагаемся на сведения клана Чёрная Птица. Сейчас мы пытаемся попасть в Командный Пост чтобы собрать больше сведений.

Тебе следует помнить об этом, если ты собираешься проникнуть в Командный пост.
