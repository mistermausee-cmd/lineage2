Вице-иерарх Мао:
Да пребудут с тобой благословения Норнил!
Если пойдёшь отсюда на юг, ты увидишь Сад Норнил, дом Норнил, Матери этого Острова Душ. Меня зовут Мао, и я ведаю этим местом. Если ты сумеешь превозмочь испытания этого места и встретить Мать Норнил, ты сможешь переродиться истинным воином-камаэлем. Будет боль, но за болью придёт радость.
