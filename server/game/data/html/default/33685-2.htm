Таинственный Дворецкий:
Этот титул властью моего господина дарован клану, из которого вышел победитель этой недели — величайший воин. Этот титул даётся лишь одному клану среди всех кланов.
Он даётся в начале новой недели клану, связанному с величайшим воином. Титул не передаётся, даже если этот воин перейдёт в другой клан.
Он даёт особый бафф, немного помогающий находящимся рядом соратникам по клану, состоящим в величайшем клане.
