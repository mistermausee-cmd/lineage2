Вервато:
Я Верватo! Моя священная задача — обманывать смерть и оживлять павших товарищей, чьи души готовы отправиться в тёмные подземные пределы.
Моя магия могущественнее даже магии колдунов племени Элроки; я величайший призыватель трупов из всех!
Но помни, что призыв трупа возможен, только если труп есть... и, конечно, при наличии необходимой платы.
