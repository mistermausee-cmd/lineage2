Верховный Заклинатель Кинсли:
Пока строилась Башня Слоновой Кости, заклинатели заключили соглашение с императором племени Кат. Невзгоды почти довели племя Кат до грани вымирания! Магистры Башни Слоновой Кости предложили переселить их в Элизиум, злой мир, при условии, что они вернутся и помогут нам, если это когда-нибудь понадобится...
