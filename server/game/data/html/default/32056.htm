Хаватари Арен:
Крепкое тело и боевую мощь орков-воинов не обрести без труда. Их можно добыть лишь самой тяжёлой тренировкой, какую только можно вообразить: Кабарт, извечная муштра. Те, кто овладеет боевым искусством Кабарта, заслужат звание Монаха.
Те, кто в совершенстве постигнет высшее искусство Кабарта — движение тела и его связь со вселенной, — удостоятся звания Тирана. Есть и ещё более высокий ранг, но он давно запечатан и больше не присваивается. Его силы давно забыты...
