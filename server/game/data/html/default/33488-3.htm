Великаны использовали способ передвижения под названием Сайюн. Он позволял им преодолевать огромные расстояния, перепрыгивая из одной точки в другую с невероятной скоростью.
Пройдя Пробуждение, ты теперь должен обладать силой использовать Сайюн.
Более того, позади меня есть место, позволяющее использовать Сайюн. Встань на него, и должен появиться значок, который можно нажать. Эта точка Сайюна связана с той, что в Деревне Острова Талкинг.
