Таинственная торговка красками Диания:
Эй ты, искатель приключений! Да, я о тебе.
Быть может, ты ищешь качественные краски, которыми можно окрашивать символы? У меня их полно. Есть только одна загвоздка: если ты купишь одну, она будет выбрана случайно из моего ассортимента.
Никакого выбора. Если повезёт, ты получишь невероятную сделку на очень редкую краску. Если не очень повезёт — получишь обычную сделку на заурядную краску. Готов рискнуть?
Что скажешь? Чувствуешь себя везучим сегодня?
