Лекарь Чёрной Наковальни:
Что ж, есть три вида яиц: красные, жёлтые и синие. На каждый вид яиц нужно использовать репеллент подходящего цвета. Не хочу хвастаться, но у Чёрной Наковальни есть новый репеллент, который оказался весьма эффективным... по крайней мере против яиц. С самим червём — совсем другая история.
