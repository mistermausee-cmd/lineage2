Префект Карукия:
Мы, орки Атука, — Скипетр Волка! Волк — наш брат и наш страж, и уже долгое время он является символом нашего племени.
У других племён орков тоже есть уникальные символы. Племя Хестуи носит символ Медведя, племя Неруга — символ Летучей Мыши, племя Уруту — символ Черепа Орка, Дуда-Мара — символ Паука, а племя Ганди — символ Леопарда.
