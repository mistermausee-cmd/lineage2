Серая Колонна Кантарубис:
Мерцание — это студенистый сгусток, по сути затвердевшая жизненная энергия. Его можно найти только в этом мире.
Мерцание означает жизнь, годится в пищу, а иногда его обменивают, как деньги. Любопытная штука.
Раньше его можно было купить у фей, но в последнее время с феями творится что-то странное, и теперь достать мерцание для покупки стало куда труднее.
