Заклинательница Басиллия
Приветствую, путник. Я Басиллия, Заклинательница Башни Слоновой Кости. Рада тебя видеть.
Эта местность — там, где я родилась... Меня не было тридцать лет, и теперь всё выглядит совсем иначе. Всё, что здесь осталось, — это шрамы, оставленные войной с Грацией... Прекрасный городок моего детства исчез без следа...
