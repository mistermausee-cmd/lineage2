# Lineage 2 — Grand Crusade (соло-сборка на L2J Mobius)

Персональный сервер Lineage 2 хроник **Grand Crusade** на движке **L2J Mobius**, доработанный
и сбалансированный **под одиночную игру (соло)**. Собран из официальных исходников Mobius,
с изменёнными рейтами, ослабленными рейд-боссами, соло-доступом во все инстансы и
русифицированным кастом-меню (Community Board, alt+B).

> Полный список изменений баланса — в файле **[BALANCE.md](BALANCE.md)**.

## Установка

### 1. База данных
1. Установи MySQL/MariaDB, запусти сервер БД.
2. Создай базу:
   ```sql
   CREATE DATABASE l2jmobius DEFAULT CHARACTER SET utf8mb4;
   ```
3. Проверь доступ в `server/game/config/Database.ini` (по умолчанию: `localhost`, логин `root`, пароль пустой).
   Тот же логин/пароль используется логин-сервером.
4. Импортируй структуру и данные с помощью установщика:
   - Windows: запусти `server/db_installer/DatabaseInstaller.jar` (двойной клик или `java -jar DatabaseInstaller.jar`).
   - Linux: `cd server/db_installer && java -jar DatabaseInstaller.jar` (нужен GUI) либо примени SQL из папки `sql/`.
   В установщике укажи те же реквизиты БД и выполни установку игровой и логин-базы.

### 2. Сетевые настройки
- **Локальный тест (тот же ПК):** ничего менять не нужно — уже стоит `127.0.0.1`.
- **VPS (Azure и т.п.):** переименуй `server/game/config/default-ipconfig.xml` в `ipconfig.xml`
  и укажи в атрибуте `address` **публичный IP** сервера. Открой порты **2106** (login) и **7777** (game).

### 3. Клиент и папка System
- Нужен клиент **Grand Crusade** (протокол **110**).
- Для русского языка: папка **System Grand Crusade RU, протокол 110** (например с ресурса l2clientdev / m0nster.io).
- В `l2.ini` (внутри System) пропиши IP сервера: для локального теста `127.0.0.1`, для VPS — публичный IP.
- Русские названия предметов/скиллов/NPC и системные сообщения берутся из **RU-клиента**.

### 4. Геодата (важно!)
В сборке геодата **не входит** (папка `server/game/data/geodata` пустая).
Без неё мобы «проваливаются» и криво ходят. Скачай геодату под Grand Crusade
(формат `.l2j`) и распакуй в `server/game/data/geodata/`, затем проверь, что в
`server/game/config/GeoEngine.ini` включён `PathFinding`.

## Запуск

### Windows (в один клик)
Дважды кликни **`START_Windows.bat`** в корне — откроются два окна (Login и Game).
Дождись в окне Game строки о полной загрузке — можно заходить в игру.

### Linux
```bash
chmod +x start_linux.sh
./start_linux.sh
```

## Первый вход и права GM

1. **Аккаунт** создаётся автоматически при первом входе (введи логин и пароль в окне клиента).
2. **Сделать своего персонажа GM/админом** — выполни в базе (замени `ИМЯ` на имя персонажа):
   ```sql
   UPDATE characters SET accesslevel = 8 WHERE char_name = 'ИМЯ';
   ```
   Уровень доступа `8` = полный админ (см. `server/game/config/AccessLevels.xml`).
   В игре набери `//help` для списка админ-команд.
