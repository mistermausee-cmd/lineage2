## 2. Метод перевода (ВАЖНО для будущих правок/довода)

Переводится ТОЛЬКО видимый текст, функциональные атрибуты сохраняются 1-в-1:

**Переводим:** текст в `<title>`, текст-узлы между тегами, подписи кнопок (текст внутри
`<button>...</button>`, `<Button>...</Button>`, `<a action="...">...</a>`), текст в `<font>...</font>`.

**НЕ трогаем (иначе ломается функционал):**
- `action="bypass ..."` целиком — в частности `npc_%objectId%_...`, `admin_xxx`, `Script`, `Quest`,
  `Chat N`, номера мультиселлов/байлистов, `admin_move_to <коорд>`, id внутри команд.
- `%objectId%` и все `%плейсхолдеры%`; `fstring`/`&$NNN;` и числовые id.
- Ресурсы/атрибуты: `back=`, `fore=`, `src=`, `ICON=`, `ALIGN=`, `width=`, `height=`, `msg=`.
- HTML-структуру и переносы строк (CRLF/LF сохраняются как в оригинале).
- Кнопки/ширины: подписи держим КОРОТКИМИ (кнопки фикс. ширины — длинный текст обрежется).

**Проверка после любой партии (все должны давать пустой вывод):**
```bash
grep -rlP 'action="[^"]*[а-яА-Я]' <папка>        # кириллица в action — НЕ должно быть
grep -rlP '%[а-яА-Я_]+%' <папка>                 # переведённые плейсхолдеры — НЕ должно быть
# + сверка числа <button/<Button/<a action/action= с git HEAD (0 расхождений)
```

## 3. Источник L2Script ru (ветка `system`) — вывод

В ветке `system` лежит `l2scripts/` — сборка **L2Script (движок GoD)** с папкой русских HTML
`dist/gameserver/data/html/ru/` (~13039 файлов). Пересечение id NPC с Mobius: **1801 из 2042**.

⚠️ **Файлы КОПИРОВАТЬ НЕЛЬЗЯ** — движки разные, бипассы несовместимы:
- Mobius: `action="bypass npc_%objectId%_showTeleports"`, `npc_%objectId%_multisell 002`.
- L2Script: `action="bypass -h npc?teleport_list_1"`, `npc?multisell 518`, плюс директивы `#if/#end`
  и `${VARIABLE}`, которых в Mobius НЕТ.
Копирование сломало бы кнопки/номера мультиселлов.

➡️ **L2Script ru пригоден ТОЛЬКО как текстовый эталон** (официальные РУ-формулировки) при переводе
Mobius-файлов с сохранением родных Mobius-бипассов. Доступ к дереву: fetch ветки system через
Powers github (`pull_repository` branch=system) → работа через `git show FETCH_HEAD:l2scripts/...`
(прямой git к gateway без авторизации не проходит).
